# 🚀 QuickAI<br>

QuickAI is a full-stack AI-powered SaaS application that helps users generate content using modern AI tools.<br>
Users can create articles, blog titles, images, and get AI-powered resume reviews.<br>

---

## ✨ Features<br>

📝 AI Article Generator<br>
📰 AI Blog Title Generator<br>
🎨 AI Image Generator<br>
🪄 Remove Background from Images<br>
✂ Remove Objects from Images<br>
📄 AI Resume Review (PDF Analysis)<br>
🌎 Community Image Sharing<br>
❤️ Like System for Community Creations<br>
🔐 Secure Authentication with Clerk<br>
☁ Image Hosting using Cloudinary<br>
